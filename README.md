用于快速搭建一个流媒体开发基础环境

#### 快速开始（Linux）：
~~~sh
clone https://github.com/HFauto/zxMediaKit
cd zxMediaKit
mkdir build && cd build && cmake .. && make -j && make install -j
~~~
